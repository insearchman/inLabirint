Общее
	Вид: Диаблоподобное. (возможно сделать чтобы камера поворачивалась за игроком)
	Мультиплеер: да
	
Суть
	Карта:
	- процедурно генерируемая. генерируется в начале, но собирается из блоков в тот момент, когда на них посмотрит любой игрок
	- 4 комнаты силы. В каждой из этих комнат есть свой источник энергии: красной, синей, зелёной, жёлтой.
	- арена. На ней находятся 4 флага игроков по углам на возвышеностях.
	- сам уровень процедурно генерируется в начале, но собирается из блоков только при взгляде на еще неоткрытую стену. 
	- игрок проходя по тунелю, по которому ещё никто ещё не проходил будет знать это, как и если по нему до этого кто-то уже прошёл.
	- каждый класс генерирует свой тип блоков
	- если между игроками одной команды разных классов меньше (например 10) блоков, то создаваемые блоки смешиваются.
	- флаги на арене расположены в разных углах.
	- рядом с флагами есть укромные места для засады.
	- можно поднимать свой флаг, а также можно опускать вражеские флаги.
	- при первом взаимоействии с любым флагом появляется индикатор нахождения арены у всех игроков.
	
	Игроки:
	- может быть 4 или кратно 4 (максимум 64) количество игроков в одной игре.
	- каждый появляется в какой-то рандомной комнате силы.
	- имеет 4 умения, различающиеся в зависимости от класса.
	- сразу может активировать одно умение, той силы, в команте которой появился.
	
	Характеристики:
	- сила		(1)		влияет на урон игрока.
	- стамина	(10)	нужна для использования умений.
	- жизни		(100)	игрок умирает, если они закончатся.
	
	Гейм дизайн:
	- игроки изначально имеют достаточно много жизней (100), и достаточно слабую силу (1), чтобы в процессе убийства другого игрока не было какого-то сильного смысла.
	- в каждой комнате силы игрок может получить способность цвета комнаты.
	- помимо способности игрок получает х2 к силе.
	- любая способность должна быть озвучена криком или возгласом, чтобы увеличить эпичность сражений
	
	Цель игры:
	- главная: поднять свой флаг на арене.
	- альтернативная: убить игроков других команд.
	- дополнительная: получить очки за количество открытых блоков тунеля.
	- дополнительная: выиграть без определённой силы.
	
	
	
	Классы:
	- Убийца
		1. жёлтый:	может видеть направление открытия тунеля
		2. синий:	может быстро бегать
		3. зелёный:	может установить ловушку
		4. красный:	может оглушить ударом с разбегу
		
	- Разведчик
		1. жёлтый:	может видеть дальше
		2. синий:	может пускать световой шар, открывающий тунель
		3. зелёный:	
		4. красный:	может пускать огненный шар, наносящий урон на расстоянии
		
	- Танк
		1. жёлтый:	может увеличивать стамину
		2. синий:	может увеличивать силу (х2)
		3. зелёный:	может увеличивать жизни (+30)
		4. красный:	может впадать в ярость, кратковременно (х2 силы на 3 секунды) увеличивая силу, стамину, жизни
		
	- Монах
		1. жёлтый:	
		2. синий:	может быстро поднимать флаг
		3. зелёный:	может отхилить жизни себе или члену команды
		4. красный:	может бить посохом с размуха, сразу несколько врагов по конусной траектории
